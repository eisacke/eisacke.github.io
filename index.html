Biliardino

This was a group project that I undertook with my fellow teaching assistants at General Assembly. The app allows users to find all of the foosball tables accross London, and submit new suggestions via Google Places. Built on Node Express & Mongo, with Google Maps & Google Places APIs.

Planit

An app that allows users to create beautifully mapped event itineraries that can be emailed out to guests. This was my final project at GA. I had full autonomy over what to build, and which tech to use. Built on Node & Express, Angular with the Google Maps & Google Places APIs, and a front end in Angular.

Launch project View code on Github

Brew

An app that maps user recommended rooftop bars across London. This was a simple yet polished group project, built over 4 days. My role was to build the API that would store the bar data and return it to the front-end, as well as assist with layout and styling. An API built on Node & Express, with the Google Maps API, Weather Underground API, Google Geocoding API, with a front end built in JavaScript.

Launch project View code on Github

Skillstack

An app that creates a community of users who want to share their skills and learn from each other. A user can post, browse and book lessons, and manage their account and bookings, as both a student and as a teacher. An app built with Ruby on Rails, Ransack, Geocoder, Carrierwave, Rmagick, Bcrypt.
